수리 완료 후 고객이 확인할 작동 마감 체크리스트

이번 작업의 확인 포인트

불편이 생긴 위치와 부속 상태를 확인하고 주변 마감재를 보호하면서 필요한 조정, 체결, 교체 작업을 진행합니다.

'마감 주변에 잔여물이나 날카로운 부분이 남음' 상황에서는 임의로 손대거나 사용을 반복하기 전에 현재 위치를 사진으로 남기는 편이 좋습니다. 수리가 끝난 직후 외관만 보고 넘어가면 반복 작동에서 생기는 문제를 놓칠 수 있는 상황의 범위를 정하려면 '부속의 체결과 정렬', '반복 작동 시 움직임', '주변 마감과 청소 상태' 세 항목을 한 흐름으로 살펴야 합니다.

여기서는 작업 완료를 단정하지 않고 '작동은 되지만 간격과 수평이 달라짐' 상태를 안전하게 확인하는 방법만 다룹니다. 세븐홈케어 상담 자료에는 '사용 시 주의사항'까지 포함해야 화면 밖 조건 때문에 판단이 달라지는 일을 줄일 수 있습니다.

1. 먼저 구분할 증상

확인할 대표 증상은 1) 한 번은 정상인데 반복하면 다시 걸림. 2) 작동은 되지만 간격과 수평이 달라짐. 3) 마감 주변에 잔여물이나 날카로운 부분이 남음. '한 번은 정상인데 반복하면 다시 걸림'은 전체 위치 사진으로 기록하고, '마감 주변에 잔여물이나 날카로운 부분이 남음'은 근접 사진과 치수 사진으로 나눠 남깁니다. '처음과 끝까지 같은 힘으로 움직이는지'와 '사용 시 주의사항'을 같은 촬영 방향과 거리에서 기록해 서로 다른 조건이 섞이지 않게 비교합니다.

증상을 기록할 때는 첫 자료를 '작업 전 전체', 다음 자료를 '반복 작동 영상' 순서로 남기는 것이 좋습니다. '한 번은 정상인데 반복하면 다시 걸림'이 보인다면 대상을 움직이거나 분해하지 않고 전체 위치와 근접 상태를 같은 방향에서 기록합니다. '마감 주변에 잔여물이나 날카로운 부분이 남음'처럼 추가 손상 가능성이 있는 상태는 억지로 반복하지 말고 정지 사진으로 현재 위치를 기록합니다.

2. 구조를 함께 봐야 하는 이유

이 문제를 이해할 때는 부속의 체결과 정렬, 반복 작동 시 움직임, 주변 마감과 청소 상태를 서로 무관한 항목으로 보지 않는 것이 중요합니다. '부속의 체결과 정렬', '반복 작동 시 움직임', '주변 마감과 청소 상태' 세 확인 항목을 함께 보아야 '한 번은 정상인데 반복하면 다시 걸림'과 '마감 주변에 잔여물이나 날카로운 부분이 남음'의 원인을 구분할 수 있습니다.

부속의 체결과 정렬 전체가 보이는 작업 사진과 반복 작동 시 움직임 근접 사진을 번갈아 비교합니다. 첫 화면에서는 설치 방향을, 다음 화면에서는 마모·이탈 여부를 확인하고, 주변 마감과 청소 상태 주변 간격까지 이어 보면 '작동은 되지만 간격과 수평이 달라짐'이 생기는 구간을 찾기 쉽습니다.

사진에서 이어서 볼 부분

작업 전 증상과 문제 부위, 사용한 부속, 완료 후 작동 상태가 사진에서 이어지는지 확인해 보세요.

3. 사진과 영상으로 확인할 순서

상담 전에 확인할 순서는 1) 처음과 끝까지 같은 힘으로 움직이는지. 2) 10회 반복해도 소음과 걸림이 없는지. 3) 좌우 간격과 수평. 4) 체결부 흔들림. 5) 주변 오염과 날카로운 면. 6) 사용 시 주의사항. '처음과 끝까지 같은 힘으로 움직이는지'에서 전체 구조를 잡고 '사용 시 주의사항'에서 주변 조건을 마무리하면 문제 지점과 실제 크기를 같은 방향으로 비교할 수 있습니다.

촬영은 '처음과 끝까지 같은 힘으로 움직이는지' 항목을 한 화면에서 비교할 수 있도록 시작합니다. 그다음 '10회 반복해도 소음과 걸림이 없는지' 부분을 가까이 찍고, 치수가 필요하면 자의 시작점과 끝점이 함께 보이게 합니다. 영상이 필요한 경우에는 '작업 후 정면' 형식으로 한 번만 천천히 촬영해 문제 구간이 화면 밖으로 벗어나지 않게 합니다.

4. 직접 확인할 때 피해야 할 행동

상태 확인 과정에서는 1) 작동 한 번만 보고 확인 종료. 2) 강한 힘으로만 테스트. 3) 완료 사진 없이 부속 위치를 기억에만 의존. 같은 행동을 피해야 합니다. 특히 '작동 한 번만 보고 확인 종료' 같은 행동은 부속의 체결과 정렬, 반복 작동 시 움직임 양쪽에 동시에 힘을 전달해 기존 손상 범위를 넓힐 수 있습니다.

분해가 필요한지 확실하지 않다면 '작업 전 전체'와 '교체 부속' 두 자료까지만 준비하는 편이 안전합니다. 이미 '강한 힘으로만 테스트' 조치를 했다면 사용한 재료와 위치를 상담 때 알려야 '작업 대상 종류' 조건을 가리지 않고 확인할 수 있습니다.

5. 작업 범위가 달라지는 조건

현장에서 작업 범위를 정할 때는 1) 작업 대상 종류. 2) 부속의 조정 범위. 3) 기존 레일과 문짝 상태. 4) 마감재 보호 범위. 항목을 함께 봅니다. 특히 '작업 대상 종류'와 '마감재 보호 범위'가 달라지면 사진상 모양이 비슷해도 준비물, 접근 방법, 작업 순서가 달라질 수 있습니다.

사진 상담에서는 '작업 대상 종류'와 '마감재 보호 범위' 두 항목을 우선 구분해 준비 방향을 안내합니다. 반복 작동 시 움직임의 규격이나 내부 상태가 사진에 보이지 않으면 추측하지 않고, 실제 치수와 접근 공간을 확인한 뒤 최종 범위를 정합니다.

6. 설명 이미지에서 볼 부분

작업 이해를 돕는 설명 이미지 - 완료 후 네 가지 검수 항목을 한 화면에서 확인합니다.

설명 이미지의 화살표와 강조 영역은 부속의 체결과 정렬, 반복 작동 시 움직임, 주변 마감과 청소 상태의 관계를 쉽게 구분하기 위한 표시입니다. '작업 이해를 돕는 설명 이미지 - 완료 후 네 가지 검수 항목을 한 화면에서 확인합니다' 자료는 부속의 체결과 정렬 구조 이해에만 사용하고, 실제 위치·규격·작업 결과는 관련 작업 사진과 현장 상태로 다시 확인합니다.

7. 완료 후 반복해서 확인할 항목

작업 후에는 1) 저속과 보통 속도. 2) 완전 열림과 닫힘. 3) 반복 작동. 4) 완료 사진과 주의사항 공유. 항목을 확인합니다. '저속과 보통 속도'와 '완료 사진과 주의사항 공유'는 작업 전과 같은 촬영 방향에서 확인하고, 전체 상태와 근접 마감이 모두 기록됐는지 비교합니다.

완료 사진은 작업 전과 같은 방향에서 촬영해 '저속과 보통 속도' 및 '완전 열림과 닫힘' 항목의 변화를 비교합니다. 교체하거나 조정한 부속이 있다면 위치를 함께 기록하고, 이후 '완료 사진과 주의사항 공유' 문제가 다시 생기면 같은 구간을 촬영해 이전 상태와 비교합니다.

8. 상담 전에 준비할 자료

빠른 확인을 위해 1) 작업 전 전체. 2) 교체 부속. 3) 작업 후 정면. 4) 반복 작동 영상. 자료를 준비해 주세요. 작업 전 전체에는 문제 부위의 전체 위치가, 교체 부속에는 부속의 체결과 정렬 또는 손상 경계가 선명하게 보여야 합니다. '반복 작동 영상'을 촬영할 때 개인정보, 얼굴, 주소, 차량번호가 함께 잡히면 가린 뒤 전달합니다.

세븐홈케어는 전달받은 자료에서 '처음과 끝까지 같은 힘으로 움직이는지' 항목과 '사용 시 주의사항' 항목을 먼저 대조한 뒤 가능한 작업 범위와 추가 확인 사항을 정리합니다. '수리 완료 후 고객이 확인할 작동 마감 체크리스트' 상담에서도 현장 조건이 사진과 다르면 확인된 내용에 맞춰 범위를 조정하고, 보이지 않는 원인이나 결과는 단정하지 않습니다.
