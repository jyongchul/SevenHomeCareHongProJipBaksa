☑️ 타공 전, 준비 사항
💠
이중유리 여부 확인 -
일반유리, 강화유리
반강화유리등 유리 재질 확인
💠
위치 파악 -
배관이 지나갈 공간과
장애물이 있는지 확인
💠
타공 준비 -
현장 확인 후 필요한
장비와 안전 조치를 끝냈습니다
사전 체크를 철저하게 하면,
작업의 안정성과 완성도가 높아집니다
사전 준비를 끝낸 후,
타공작업을 시작했습니다

🎯 타공 전, 위치 확인과 체크 사항
1️⃣ 정확한 위치 측정 :
타공 크기를
마지막으로 정확하게 확인하고
손상 위험도가 낮은 적당한 위치를
찾아 표시해 주었습니다
2️⃣ 작업 공구 준비 :
유리 타공
전용 드릴과 안전장비를 준비하고
주변을 정리해 작업공간을
마련해 주었습니다

🧩 타공전, 유리전용 필름 붙이기
1️⃣ 공구가 미끄러지는걸 방지 :
유리는 표면이 매끄러워 공구가
미끄러워 흔들리기 쉬운데
전용필름이
미끄러지는걸 막아 안정적인
타공 작업이 가능
합니다
2️⃣
작업 중 유리의 파손 감소 :
전용필름이 작업시 가해지는
마찰을 줄여주어
타공시 미세한
균열이 생기는걸 방지
해 줍니다

✨ 1차 내측 유리 타공 완료
내측 유리에 손상이나
미세 균열이
생기지 않고 깔끔하게 타공
되었습니다
유리가 과열되지 않게 충분히
냉각수를 공급
해주고,
각도와 압력을 조절
해 주며 작업해
균열이 생기지 않게 주의
했습니다

📌 주의했던 핵심 포인트
→ 수평 각도 유지 :
드릴이
유리면과
수직이 되어 드릴링 되도록
각도 조절
해주었습니다
→ 저속 드릴링 :
속도가 너무 빠르거나
압력이 세게 가해지게되면
균열로 이어질 수 있기 때문에
시간이 걸리더라도 천천히 작업
했습니다
→ 적절한 냉각조치
: 천천히 오랜시간
드릴링 작업이 가해지면 유리가
열을 받아 약해집니다
냉각수를 계속 적셔 주면서
열이 축적되는걸 방지
해 주었습니다
✨
위와 같이 세심하고 전문적으로
작업한 덕분에 미세한 스크래치
하나 없이 완벽하게 작업되었습니다

🛡️ 외측유리 타공시, 주의할 점
→ 먼저 작업한,
내측 유리를
손상시키지 않도록 세심하게 주의 :
타공시, 공구가 안쪽 유리면에
직접 닿지 않도록
각도를 잘 유지
하고,
압력을 분산시키는것이 중요
합니다
→ 정확하게 위치를 조율 :
내측 유리 타공과 외측 유리 타공의
위치가 정확히 일치해야 합니다
수평이 맞아야, 배관이 깔끔하게 통과하고
유리 성능 유지에 문제가 없습니다
