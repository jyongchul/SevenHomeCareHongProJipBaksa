오늘 포스팅해드릴 현장은
위례신도시에 위치한
아파트였습니다
롤러가 파손되고 분실된
붙박이장 슬라이딩 도어를
튼튼하게 수리하고 온
오늘의 현장을 소개합니다

오늘의 현장

슬라이딩 도어가 빡빡하고
잘 움직이지 않아 고쳐보려고
도어를 분리하시다가 롤러가 깨지고
분리되 급히 연락을 주셨다고 합니다

준비해간 새 롤러로
튼튼하게 수리를 끝낸 모습입니다
그럼 오늘의 수리과정
포스팅을 시작하겠습니다

홍프로 집박사 문의하기
사진을 누르면 전화연결됩니다

작업중에는 소음과 작업집중을 위해
통화가 어려운 경우가 많습니다
자세한 현장 사진을 남겨주시면
적합한 시공 방법과 견적을
안내해 드리겠습니다.

오늘 방문드린 위례신도시
현장 위치입니다
거리가 멀것 같아 고민되신다면
고민하지 마시고
언제든 부담없이 연락주세요
집수리가 필요한 곳이라면
최대한 빨리 방문드려
수리해드립니다
홍프로 출장 가능 지역
서울, 경기 전 지역
인천 검단 송동 청라 안산 배곧 위례
시흥 구리 일산 성남 분당 안양 송파
서초 반포 양재 구로 하남 수원 광교
영통 동탄 용인 기흥 수지 흥덕 판교
한숲 안성 평택 천악 아산 고덕 향남
외 지역도 방문 드리고 있습니다
위 지역외에도 방문드립니다

1. 작업내용 : 붙박이장슬라이딩도어 롤러수리
2. 작업인원 : 1명
3. 작업시간 : 1시간 30분
4. 해결방법 : 새 롤러로 교체 수리 완료

1단계_현장 체크하기

잘 사용하시던 붙박이장이 갑자기
구동이 잘 안되고 뻑뻑한 느낌이 들어
직접 수리해 보시려고
도어를 분리하시다가
도어가 너무 무거운탓에
롤러가 파손되었다고 하십니다

분리하던중 상부롤러가 깨지고
하부부품은 분리되어
도어를 한쪽에 세워두신 상태였습니다

그럼 분리된 롤러는 재조립해드리고
깨진 롤러는 튼튼한 새롤러로
교체해 드리겠습니다

2단계_파손된 롤러 교체 수리

상부 우측에 조립된 롤러는
파손되어 분실된 상태였습니다

상부 좌측에 조립된 롤러도
반정도 깨져 교체가 필요한 상태였습니다

준비해간 롤러중 기존 롤러와
동일한 규격의 부품으로
교체드렸습니다

좌측 우측 새 롤러로
교체완료 후 롤러상태를
체크해 봤습니다
롤러가 베어링과 잘 조립되어
구동도 잘되는 상태입니다
마지막으로 기존 구멍자리에
보강작업을 추가로 진행해서
튼튼하게 조립해 드리겠습니다

3단계_피스 구멍 보강 작업

부품이 조립되있던 자리는
시간이 지나면서 커지기도하고
분리하는 과정에서도 구멍이 커집니다
때문에 기존자리에 목심을 박아서
튼튼하게 보강작업을 해드리고 있습니다

헐거워진 자리에 재조립을 할 경우
부품이 떨어지고 분리되는 경우가
생길 수 있기 때문에
보강작업을 꼼꼼하게 진행해야 합니다

4단계_상부/하부 부속품 조립

보강작업을 완료한 자리에
상부 좌측 우측 롤러부속품을
조립해 주었습니다
여러번 당겨도 흔들림없이
단단하게 조립되었습니다

분리되었던 하부 부속품도
단단하게 조립해 주었습니다
이제 도어를 조립해 이상은 없는지
전반적으로 점검해 드리겠습니다

5단계_도어 조립 후 점검작업

상부와 하부 레일의 홈을 잘 맞춰
조립해 주었습니다
조립 후 도어구동이 잘 되는지
고객님과 여러번 확인해 보았는데
걸리는 부분없이 잘 움직여 줍니다

추가로 고객님께서
점검하고 싶으신 부분
체크해 드리며 수리를 마무리 했습니다
