거실 강마루 물먹어 변색된 곳 시트지 작업 방법

세븐홈케어

고객님 의뢰 내용입니다.
거실 마루에 카펫을 깔아놓았는데
습기가 차고 물을 먹어서 마루가 변색이 되고
심한 곳 표면이 벗겨져 있는 상태입니다.
강마루로 시공이 되어있는데 교체하기는
싫고 시트지로 시공이 가능할까요?
강마루란,
합판에 원목 무늬의 시트지를 붙여서
제작된 제품을 강마루라고 합니다.
시공 방법은 접착제를 바닥에 도포하고
마루를 완전히 밀착시켜서 붙여줍니다.
열전도율이 우수하고 꿀렁거리지 않고
디자인이 다양하며 습기에도 강한 편입니다.
주거 시설에 주로 시공되고 있으나 시공
단가가 좀 높은 편입니다.

오늘 작업 과정 소개

● 작업 내용 : 거실 마루 물먹어 변색된 곳 시트지 작업
● 작업 인원 : 1명
● 작업 시간 : 4시간 30분
● 작업 위치 : 서울 광진구 빌라

작업 과정

세븐홈케어

1. 현장 상태 확인
2. 비닐 보양제 작업
3. 강마루 표면 이물질 제거

1. 현장 상태 확인

세븐홈케어

이번 현장은 서울 광진구에 위치한 빌라
인데요
거실 마루가 물을 먹어서 변색이 되고
심한 곳은 표면이 군데군데 벗겨져 있네요.
카펫을 깔아 놓으셨는데 오랫동안 습기가
차서 관리가 안 된 것 같다고 하시네요.

2. 비닐 보양제 작업

세븐홈케어

시공을 하면서 먼지가 발생할 수 있어서
소파에 비닐 보양제 작업을 했습니다.
분진이 최소로 나오도록 작업을 하지만
보양제 작업은 반드시 해야 합니다.

3. 강마루 표면 이물질 제거

세븐홈케어

강마루 표면이 물을 먹어서 변색도 되었지만
표면도 거칠고 스크레이퍼로 살짝만 문질러도
이물질이 쉽게 떨어져 나옵니다.
면상 태가 좋지 않은 상태에서 시트지 작업을
하게 되면 밀착이 제대로 되지 않아서 들뜨기
쉽게 되어
반드시 면을 고르게 작업을
해주어야 합니다.

4. 면 보강 작업

세븐홈케어

강마루 표면 이물질을 1차로 제거한 뒤에
마루가 페인 곳을
전용 보강제를 바르고 나서
일정 시간 동안 굳히고 나서 사포로 면을
반듯하게 만들어 주었습니다.
페인 곳은 상태에 따라서
2차 작업까지 진행
해서 면을 최대한 고르게 만들어 주어야 합니다.
움푹 파인 곳에 시트지를 붙이게 되면 시공 후
퀄리티가 떨어지게 됩니다.

5. 시트지 작업

세븐홈케어

시트지는
실측 진행시 샘플북을 보면서
현재 마루와 가장 유사한 시트지로 준비
를
해서 시공을 해드리고 있습니다.
같은 시기에 구입한 마루라도 사용감이 있는
마루와 사용하지 않고 보관해둔 마루는
색감이 차이가 나기 마련입니다.
시트지도 마루와 완전히 동일한 색상을
구할 수는 없지만
시공을 했을 때 이질감이
가장 적은 색상을 선택하여 시공을 해드리기
때문에
고객님께서도 아주 만족을 하십니다.
면 작업을 완료하고 나서 수성 본드를 시트지
시공할 면에 골고루 바르고 나서 마를 때까지
시간을 두고 기다려 주었습니다.
시트지도 접착성이 있어서 붙였을 때 쉽게
떨어지지 않지만 수성 본드를 바르고 시공을
하게 되면 접착성이 더욱 높아지게 됩니다.
강마루 폭에 맞게 시트지를 재단해서 한 폭
한 폭 시공을 하기
때문에 시공
시간은 오래
걸리지만 붙이고 나서 매지 경계면이 보여서
실제 강마루를 새로 붙인 것 같은 효과를
나타낼 수 있습니다.
강마루를 교체하려면 비용도 부담스럽지만
시공 시 발생하는 소음과 분진 본드 냄새가
많이 날 수 있습니다.
